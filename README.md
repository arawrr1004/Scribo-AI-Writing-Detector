## AI Writing Detector with Speedometer

Aplikasi Android modern untuk mendeteksi teks buatan AI (ChatGPT) menggunakan Machine Learning.

### Tech Stack:
- **Backend:** Python (FastAPI, Scikit-Learn)
- **Mobile:** Android Kotlin (Jetpack Compose, Retrofit)
- **AI Model:** Logistic Regression trained on 140k+ samples.

### Features:
- **Speedometer Gauge** interaktif untuk skor probabilitas AI.
- **Real-time API connection** dari Android ke Python.

### Folder Structure:
- `/Backend-Python`: Server FastAPI dan model AI (.pkl).
- `/app`: Source code aplikasi Android.
